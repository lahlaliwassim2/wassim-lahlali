Les commandes qu’ on a appris cette séance a propos de git et GitHub

Commande de configuration pour lier notre compte GIT et notre compte GITHUB :
- git config --global user.name "Votre nom"
- git config --global user.email votre-email@email.com
Commande pour Vérifier si l'identification s'est effectuer correctement :
-git config --get user.name
-git config --get user.email
Commande pour savoir le code ssc :
-ls ~/.ssh/id_rsa.pub
Commande qui ajoute tous les fichiers modifiés :
-Git add (fichier) 
Commande publie les changements dans l’arbre git local :
-git comit 
Commande qui télécharge en local les modifications présentes sur le serveur :
-git pull 
Commande qui pousse sur le serveur les modifications locales :
-git push
